#### :x:风险概述:

本工具旨在帮助企业快速定位漏洞修复漏洞,仅限授权安全测试使用!

严格遵守《中华人民共和国网络安全法》,禁止未授权非法攻击站点!



#### :ballot_box_with_check:使用教程

将测试站点保存在当前目录下的url.txt

字典文件ueditor.properties默认有常见漏洞路径，用户可以自定义添加扩展

命令行输入:java -jar UEditorGetShell.jar
